Commonplace Book
================

A commonplace book written in Markdown.

[![License][license-image]][license-url]

Usage
-----

Edit `contents.txt` and add or remove quotes as desired.

To view the book, open `index.html` in a browser.

### Command line

To edit the book:

    $ npm run edit

To view the book:

    $ npm start

To run tests:

    $ npm test

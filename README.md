Commonplace Book
================

A commonplace book written in Markdown.

Usage
-----

Edit `contents.txt` and add or remove quotes as desired.

To view the book, open `index.html` in a browser.

### Command line

To view the book:

    $ npm start

To run tests:

    $ npm test
